歡迎使用Pygame Zero
======================


Pygame Zero可以讓人們輕鬆無障礙創建遊戲。

它是出於教育目的開發的，所以老師們可以用pygame zero講授基本的程式知識，同時又不需
要解釋複雜的Pygame的API和事件循環。

歡迎關注公眾號，少兒創客幫，或者加微信 ilangxm 加入 pgzero 討論群。
